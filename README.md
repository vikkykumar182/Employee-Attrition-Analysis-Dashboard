# 📊 Employee Attrition Analysis Dashboard

An interactive **HR Analytics Dashboard** built using **Power BI** to analyze employee attrition, workforce demographics, salary distribution, and departmental performance. The dashboard enables HR professionals and business stakeholders to identify workforce trends and make data-driven decisions.

---

## 🚀 Project Overview

This project provides a comprehensive analysis of employee attrition by visualizing important HR metrics and KPIs. It helps organizations understand employee turnover patterns based on department, job role, salary, education, age, and years at the company.

---

## 🛠️ Tools & Technologies

- Power BI
- DAX
- Power Query
- Microsoft Excel

---

## 📌 Dashboard Features

- Employee Attrition Analysis
- Workforce Demographics
- Department-wise Attrition
- Job Role Analysis
- Salary Slab Distribution
- Education-wise Attrition
- Age Group Analysis
- Years at Company Analysis
- Interactive Filters & Slicers
- KPI Cards

---

## 📈 Key Performance Indicators (KPIs)

- Total Employees
- Attrition Count
- Attrition Rate
- Average Salary
- Average Age
- Average Years at Company

---

## 📊 Dashboard Insights

- Analyze employee attrition across departments.
- Identify job roles with the highest attrition.
- Compare attrition by salary slabs and education levels.
- Explore workforce demographics through interactive charts.
- Support HR decision-making with dynamic visualizations.
